# Data Structures & Algorithms Toolkit (C++)

## Description
This is a command-line based project where core data structures and algorithms are implemented from scratch using C++.  
The main goal of this project is to strengthen fundamental computer science concepts.

## Features
- Stack (Array-based)
- Queue (Array-based)
- Linked List (Singly)
- Searching Algorithms
- Sorting Algorithms
- Binary Search Tree

## Technology Used
- Language: C++
- Programming Style: Object Oriented
- Interface: Command Line (CLI)

## How to Run
g++ src/*.cpp -Iinclude -o dsa  
./dsa
